# T10读卡器配置工具 使用说明

**版本**: 1.0.0.1  
**更新日期**: 2026-03-20

---

## 一、软件简介

本工具用于配置 T10 系列读卡器的射频参数，包括：

- 射频通信速率设置 (106K/212K/424K/848K)
- WTX (等待时间扩展) 次数设置
- 恢复设备默认参数

### 支持连接方式

- **USB 连接** (端口 100)
- **串口连接** (COM1-COM16, 波特率 9600-230400)

## 四、快速开始

### 1. 连接设备

- 将 T10 读卡器通过 USB 线或串口线连接到电脑
- 运行 `T10SetTool.exe`

### 2. 打开设备

在 **Connection Settings** 区域选择连接类型：

- **USB**: 选择 `USB (port=100)`
- **串口**: 选择 `Serial (COM)`，然后选择对应的 COM 口号和波特率

点击 **Connect** 按钮连接设备。连接成功后，**FW Ver** 框会显示固件版本号。

### 3. 配置射频参数

在 **RF User Attributes** 区域选择操作类型：

| 操作类型 | 说明 |
|----------|------|
| type=0x01 Set RF Rate | 设置射频速率 |
| type=0x02 Get RF Rate | 读取当前射频速率 |
| type=0x03 Set WTX Count | 设置 WTX 次数 |
| type=0x04 Get WTX Count | 读取当前 WTX 次数 |

#### 设置射频速率

1. Operation 选择 `type=0x01 Set RF Rate`
2. RF Rate 选择目标速率 (106K/212K/424K/848K)
3. 点击 **Set Param** 按钮

#### 设置 WTX 次数

1. Operation 选择 `type=0x03 Set WTX Count`
2. 在 WTX Count 输入框输入数值 (建议 1-255)
3. 点击 **Set Param** 按钮

### 4. 恢复默认参数

点击 **Restore Default** 按钮，将所有射频参数恢复为出厂默认值。

### 5. 诊断设备支持的功能

点击 **Probe All** 按钮，工具会自动测试所有操作类型。测试结果会显示在日志区，可查看哪些功能被固件支持。

### 6. 断开连接

- 点击 **Disconnect** 按钮安全关闭设备
- 或直接关闭程序窗口

## 六、日志信息说明

| 标记 | 含义 |
|------|------|
| `[OK]` | 操作成功 |
| `[ERR]` | 操作失败，请检查连接和参数 |
| `[WARN]` | 警告信息，操作可能部分成功或不被支持 |

### 常见错误码

- **ret=-2** - 该功能不被当前固件版本支持 (常见于 Get 操作)

## 七、注意事项

### 1. 参数保存

> ⚠️ **重要**: 通过本工具设置的参数在设备断电后会丢失！
>
> 每次上电后需要重新配置，或在应用程序中设置。

### 2. 固件兼容性

部分 T10 固件版本可能不支持 Get 操作 (type=0x02/0x04)。如遇 `ret=-2` 错误，说明该功能不被支持，请使用 **Probe All** 检测。

### 3. 射频速率选择

| 速率 | 说明 |
|------|------|
| 106K | 标准速率，兼容性最好 |
| 212K | 较高速率 |
| 424K | 高速率 |
| 848K | 最高速率，部分卡片可能不支持 |

### 4. WTX 次数

- 用于延长卡片响应等待时间
- 一般保持默认值即可，仅在特殊卡片需要时调整
